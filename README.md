# Phishing-URL-Detector

## Objective

This project detects whether a URL is safe or potentially phishing by analyzing URL structure, suspicious keywords, HTTPS usage, and domain age.

## Features

* HTTPS Verification
* Suspicious Keyword Detection
* IP Address Detection
* Domain Age Analysis
* Risk Scoring System
* SAFE / WARNING / DANGEROUS Classification

## Technologies Used

* Python
* Requests
* Python-WHOIS
* Regex

## Installation

pip install python-whois

## Run

 python phishing_detector.py

## Example

Input:
https://google.com

Output:
SAFE
